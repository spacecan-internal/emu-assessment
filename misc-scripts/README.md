# Misc scripts

This directory contains various scripts that are useful for development and testing.
Could be used for inspiration or as a starting point for your own scripts.

## List of scripts

- [**`custom-properties.sh`**](custom-properties.sh) - Pull repos custom properties into a csv file.
- [**`link-gh-groups.sh`**](link-gh-groups.sh) - Update a sheet with AD Groups to their GH team IDs.
